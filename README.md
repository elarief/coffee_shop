# coffee_shop
Coffee shop kami adalah tempat di mana aroma kopi berpadu dengan suasana hangat dan nyaman. Cocok untuk nongkrong santai, ngerjain tugas, atau sekadar menikmati waktu sendiri. Setiap cangkir diracik dengan sepenuh hati, dari biji kopi pilihan hingga sentuhan rasa yang bikin betah berlama-lama.
